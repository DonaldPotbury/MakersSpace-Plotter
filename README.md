# Makers Space-Plotter

![Test](<images/CNC Plotter Cover Photo.png>)

The inspiration for this project came from the Instructables article [Arduino Mini CNC Plotter](https://www.instructables.com/Arduino-Mini-CNC-Plotter/) published by [MertArduino](https://www.instructables.com/member/MertArduino/)   Which is beautiful in its simplicity and completeness.

I modified it slightly for my purposes in the but the concept is the same.
- Modified the geometry of the X and Y axis.  
	- Eliminate printed overhang
	- Allow more
- Modified the Z-Axis to match X and Y.  This should allow for more flexibility in tools



## 3D Parts and Assembly
Many of the components have 3.5mm holes.  Use an M4 tap to thread these.  Then connect the parts using M4x6mm pan head machine screws. 

## The Circuit
Like [MertArduino](https://www.instructables.com/member/MertArduino/) I breadboarded the circuit which was fine for a testing but wanted a circuit board to reduce the chaos.  My friend David suggested I make the board large enough to serve as the base for the entire project.  So that's what I did!  I designed the board using [EasyEDA](https://easyeda.com/) and is shared here.

## Installing the Firmware on the Arduino
This plotter uses GRBL which is a which is a popular firmware for Arduino Uno, Nano, or any other controller using the Atmega 328p chip.  It is designed to turn it into a fully capable CNC controller.

We will be using [GRBL-28byj-48](https://github.com/TGit-Tech/GRBL-28byj-48) which is a variant modified to accommodate the use of 28BYJ-48 stepper motors.

**Installation Procedure:**
1. Download the ZIP file from [GRBL-28byj-48](https://github.com/TGit-Tech/GRBL-28byj-48)
2. Follow the instructions at [How to flash (install) Grbl 1.1 onto an Arduino Uno, Nano, or Mega](https://www.circuitist.com/install-grbl-onto-arduino-uno-nano-or-mega/) starting at **"Installing Grbl on Arduino Uno, Nano, etc."**

## Controlling the Plotter
Controlling the plotter is done using [Universal Gcode Sender](https://winder.github.io/ugs_website/).  It is "A free and full featured gcode platform used for interfacing with advanced CNC controllers like [GRBL](https://github.com/grbl/grbl) , [FluidNC](http://wiki.fluidnc.com/), [TinyG](https://github.com/synthetos/TinyG), [g2core](https://github.com/synthetos/g2) and [Smoothieware](http://smoothieware.org/). Universal Gcode Sender is a self-contained Java application which includes all external dependencies and can be used on most computers running Windows, MacOSX or Linux."

It can be downloaded from https://winder.github.io/ugs_website/download/
Installation Instructions can be found at https://winder.github.io/ugs_website/installing/
